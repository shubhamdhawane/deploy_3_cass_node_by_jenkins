# Cassandra DBA Automation

Jenkins-based automation for Cassandra cluster deployment and operations.

## Features
- Cluster deployment
- Backup & restore
- Repair & cleanup
- Rolling restart
- Health checks
- TLS deployment
- Node bootstrap/decommission
- Rolling upgrades
- DR validation

## Requirements
- Jenkins
- SSH credentials
- Cassandra 4.x
- RHEL/Amazon Linux
