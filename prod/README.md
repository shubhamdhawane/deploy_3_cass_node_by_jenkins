# Cassandra DBA Automation Project

Production-style Cassandra DBA automation using Jenkins, SSH, AWS EC2, and Cassandra 4.x.

## Included
- Cluster deployment
- Health checks
- Backup & restore
- Repair
- Cleanup
- Rolling restart
- TLS deployment
- Node bootstrap/decommission
- Rolling upgrades
- DR validation

Use Jenkins credentials ID: aws-key
